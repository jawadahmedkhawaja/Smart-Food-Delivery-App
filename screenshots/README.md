## How to Take Screenshots

1. Run the app on an emulator or physical device
2. Navigate to each screen
3. Take screenshots (using emulator tools or device screenshot function)
4. Rename the files according to the list above
5. Place them in this directory

## Image Size Recommendations

- **Resolution**: 1080x2340 (or similar mobile aspect ratio)
- **Format**: PNG (for better quality)
- **Max file size**: Keep under 500KB per image for faster README loading
